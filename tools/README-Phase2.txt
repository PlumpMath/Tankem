Readme pour le construction de niveaux

#Level Builder
-Executer le fichier LevelBuilder.jar dans tools
*Il faut avoir accès à l'internet pour se connecter à Oracle
-------------------------------------------------------------------------
**Note la BD utilisé par défaut  par le Level Builder apartient à l'utilisateur e1041268
  pour modifier, modifier le l'utilisateur et le mot de passe dans la class Operations.java dans connecter().
-------------------------------------------------------------------------
-Menu en haut à droite, modifier les valeurs désirées avec les boutons.
-Naviguer avec W-A-S-D.
-Déplacer les tanks avec la touche "1" et "2" (Vert et rouge respectivement)
-Changer de tuiles avec "espace" ou "ctrl" ou cliquer.
-Choississez un nom
-Appuyer sur le bouton sauvegarder pour garder le niveau.
***Les 2 tanks doivent être placés!
****Une fois un Tank placé, on ne peut pas changer le type de case. Cependant, on peut placer
un Tank sur n'importe quel type de case. 


#Changement dans le menu du jeu
-Choix du niveau
-Seul les niveaux actifs sont affichés en ordre alphabétique.
-Le premier niveau est un niveau créé aléatoirement.
-Vous pouvez voir le status du niveau avec un "hover" de la souris.

*Si vous n'avez pas de connexion, un message d'Erreur s'Affichera au début.
 Seul le niveau hasard pourait être choisi.
 La balance des stats sera celle du defaut.

**Note la BD utilisé par défaut  pour les maps/balance appartient à l'utilisateur e1041268
  pour modifier, dans common\internal\DAO, modifier les valeurs dans Connexion
