Readme pour lecture et ecriture dans la table tankemBalance de la base de données

# Lecture de la base de données(fichierLectureBD.cmd)
	- Double-cliquer le fichier "fichierLectureBD.cmd"
	
	- Un message s'affiche dans l'invite de commande avec le résultat
	 de la connection à la BD(Succès ou echec)
	
	- Si la connection à été correctement établie, dans le selectionneur de fichier 
	  choisir ou créer un fichier CSV où mettre les données de la BD du jeu
	
	- Une fois le choix fait, le fichier s'ouvre et on peut modifier les
	 données à l'intérieur

#Ecriture dans la base de données(fichierEcritureBD.cmd)

	- Double-cliquer le fichier "fichierEcritureBD.cmd"

	- Un selectionneur de fichier s'ouvre pour sélectionner un
	 fichier CSV dans lequel prendre des données à écrire dans la BD

	- Un message s'affiche dans l'invite de commande avec le résultat
	  de l'écriture dans la BD(Succès ou erreurs possibles)
